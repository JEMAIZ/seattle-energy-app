# Seattle Building Energy Prediction 🏢

Ce projet utilise le Machine Learning pour prédire la consommation d'énergie des bâtiments non résidentiels de Seattle.

### Structure du projet :
- `app.py` : Application interactive Streamlit.
- `train_model.py` : Pipeline Scikit-Learn (XGBoost).
- `requirements.txt` : Dépendances Python.

**Modèle :** XGBoost Regressor ($R^2 \approx 0.82$)
